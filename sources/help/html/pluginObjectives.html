Form plugin

Plugin objectives

  • Executive summary
The form plugin extends the Archi behavior with the following functionalities :
  • Show up the components' metadata and properties in a graphical form
  • These metadata and properties can also be updated through the form
  • They can also be exported to an Excel spreadsheet

  • How it works
Here are the steps to follow to make the plugin working:
  1. You need to create (at least) one configuration file or use the sample provided,
  2. Declare it to the plugin through its preference page,
  3. Select one or several objects in Archi,
  4. Right click to open the context menu.
And voila: menu entries will be added by the plugin to give your access to the forms.

  • What are forms
Forms are graphical windows, spitted up with tabs, that can be configured to present widgets containing information about the selected Archi components (one form per component).

Currently, the supported widget classes are:
  • label: shows up read-only information
  • text: shows up updatable information
  • combo: shows up updatable information and provides a list of valid values
  • check box: shows up updatable binary option (like true/false, yes/no, black/white, ...)
  • table: shows up information ordered in lines and columns
